## What is DAppNode

[DAppNode](https://dappnode.com) is a platform that makes it easy to run decentralized infrastructure, including blockchain nodes and services like the CoW Protocol Watch Tower. It provides a user-friendly interface for managing and monitoring your deployments.

## Package availability

The CoW Protocol Watch Tower is available as a DAppNode package. This package is maintained in a separate repository to ensure it follows DAppNode's packaging standards and best practices.

## Installation

The watch-tower package can be installed through the DAppNode package manager. The package includes:

- Pre-configured watch-tower instance
- Web interface for monitoring
- Automatic updates
- Easy configuration management

<Note>
The DAppNode package is held in a [separate repository](https://github.com/cowprotocol/dappnodepackage-cow-watch-tower) from the main watch-tower codebase.
</Note>
